  <name>Task 2: Define Blog Post ContentModel</name>
  <files>backend/internal/modules/blog/service.go</files>
  <action>
    Define the "Blog Post" ContentModel using the hybrid schema pattern:
    - Typed Core Fields: `title` (string), `slug` (string, unique), `status` (string: Draft/Scheduled/Published), `published_at` (timestamp), `author_id` (UUID).
    - JSONB Content Fields: `body` (rich text), `excerpt` (string), `featured_image` (string), `meta_title` (string), `meta_description` (string).
    Ensure the model is registered and validated against the Core Schema Engine.
  </action>
  <verify>
    <automated>curl -X GET http://localhost:8080/api/v3/schema/models | grep "Blog Post"</automated>
  </verify>
  <done>Blog Post model is created with the correct hybrid structure</done>
</task>
